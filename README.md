# 🌤 Async Weather Tracker

## 📌 Project Overview
This project is a simple Weather Tracker built using **Vanilla JavaScript**.  
It demonstrates how asynchronous JavaScript works using **Fetch API, async/await, promises, and the event loop**.

---

## 🎯 Features
- 🔍 Search weather by city name  
- 🌡️ Displays temperature, weather, humidity, and wind  
- ⚡ Uses async/await for API calls  
- 🧠 Demonstrates event loop (microtask & macrotask)  
- 💾 Stores search history using Local Storage  
- ❌ Handles invalid city errors gracefully  

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

---

## 🔗 API Used
- OpenWeatherMap API  
  https://api.openweathermap.org

---

## 🚀 How It Works
1. User enters a city name  
2. Fetch API is used to get weather data  
3. Data is displayed on screen  
4. Previous searches are stored in Local Storage  
5. Console logs show async execution flow  

---

## 🧠 Concepts Covered
- Async/Await  
- Promises (.then / .catch)  
- Event Loop (Call Stack, Microtask, Macrotask)  
- Local Storage  
- DOM Manipulation
